A field manual for Storage Drawers — the original bulk single-item storage mod. Place a drawer on a wall, dump a stack in, and watch a live count tick up on its face. Storage Drawers MC 1.19.2 · Forge + Drawer Controller Bulk single-item storage

00 — Orientation

Start Here

Storage Drawers is the classic single-item bulk storage mod. Each drawer slot holds one item type but an enormous number of them, and prints the running count right on the block. You place a drawer, dump items in, see a live total, then grow the idea outward with upgrades and a controller until an entire wall behaves like one inventory.

core

The drawer

A wooden block with 1, 2, or 4 item slots. Right-click to deposit a stack; click the face to pull one (or a stack) back out.

count

The live count

Each slot shows its item icon and a number that updates in real time — your storage is readable at a glance, no GUI needed.

grow

Upgrades

Slot a Storage Upgrade to raise capacity; slot a utility upgrade to void overflow, emit redstone, auto-fill, and more.

network

Controller

One Drawer Controller ties every adjacent drawer in range into a single network you can pipe into and pull from.

How it differs from a chest

A chest holds many different stacks; a drawer holds one item type but far more of it, and shows the count without opening anything. Drawers shine for the materials you mass-produce — cobble, dirt, ingots, logs — where you want a thousand of one thing and instant visibility.

The shape of a build

  1. Place one drawer. Right-click to drop a stack in. The face now shows the item and a count. Click it to take items back.
  2. Quantify it. Hit it with a Drawer Key to toggle the number display on or off, base recipe needs no power.
  3. Raise capacity. Open the drawer's upgrade slots and add Storage Upgrades to multiply how much each slot holds.
  4. Add utility. A Void Upgrade discards overflow; a Redstone Upgrade reports fullness; a Filler tops things off.
  5. Wire it together. Build a wall of drawers and drop a Drawer Controller beside it — now a single hopper or pipe feeds the whole wall.
  6. Specialize. Use Compacting Drawers for metals/ores and Framed Drawers to disguise the wall as part of your build.
Where this sits in your pack

Storage Drawers is the bulk layer of a storage setup — see the Storage hub for how it slots next to chests, barrels, and full item networks. Its modern spiritual successor, Functional Storage, covers the same idea with a fresh codebase and digital-controller flavour; if your pack ships that instead, the muscle memory transfers directly. We compare them in §07.

01 — The blocks

Drawer Types

Every drawer is a wood block whose face is divided into slots. More slots means more item types in one block but less capacity each, since the block's total volume is split between them. They craft from planks of any supported wood, so they blend into whatever palette your base uses.

Basic drawer layouts
DrawerSlotsBest for
Basic Drawer (1-slot)1Your highest-volume single item — cobble, dirt, sand. Max capacity per slot.
Basic Drawer (2-slot)2A balanced default — two related items (e.g. logs + planks) in one block.
Basic Drawer (4-slot)4Sorting lots of small-volume items (dyes, seeds, nuggets) compactly.
Half Drawer (1×2)1 or 2Shorter blocks for wainscoting-height walls; same idea, smaller footprint.
Capacity is split, not added

A 1-slot drawer and a 4-slot drawer of the same wood share the same total capacity budget; the 4-slot just divides it four ways. Use 1-slot drawers for the things you have thousands of, and multi-slot drawers for variety. JEI/in-game tooltips show the exact per-slot number for your pack.

Wood variants

Drawers come in the full set of supported wood types — oak, spruce, birch, jungle, acacia, dark oak, and usually any modded woods registered through the ore dictionary/tags. They're purely cosmetic at the basic tier: pick the colour that matches the wall. (For total freedom over appearance, jump to Framed Drawers in §06.)

Reading and quantifying

By default each slot shows an item icon and a live count. The Drawer Key (sometimes called the Quantify Key in newer builds) right-clicks a drawer to toggle the number display on or off — handy when you want a clean wall of icons without numbers, or vice versa. It's a cheap held item, not an installed upgrade.

Trim — filling the wall

Trim is a non-storage wooden block in matching variants. Use it to fill gaps, frame a grid of drawers, or border a wall so the whole thing reads as one tidy panel. Trim is also where the Drawer Controller propagates its network through (see §04), so a wall framed in trim still routes correctly.

02 — Bigger numbers

Storage Upgrades

Storage Upgrades are the headline feature: drop them in a drawer's upgrade slots to multiply per-slot capacity. They come in a tier ladder, and within a single drawer the multipliers stack, so a fully upgraded drawer holds a colossal amount of one item.

Storage Upgrade tier ladder (multipliers are approximate defaults)
UpgradeRough multiplierNote
Storage Upgrade I×2The entry tier — cheap, doubles base capacity.
Storage Upgrade II×4Standard early-game bump.
Storage Upgrade III×8Mid-game; iron-tier components.
Storage Upgrade IV×16Gold-tier; serious bulk.
Storage Upgrade V×32Higher diminishing-effort tier.
Storage Upgrade VI×64Top of the numbered ladder.
Emerald Upgrade≈ ×128A premium step above the numbered tiers.
Diamond Upgrade≈ ×256Late-game; near-bottomless slots.
Obsidian Upgrade≈ ×512The ceiling tier in most configs.
Exact numbers vary by pack

The multipliers above are the typical defaults but are fully configurable, and some packs rename or rebalance the Emerald/Diamond/Obsidian tiers. Always read the upgrade's tooltip and the drawer's capacity readout in your install — the in-game numbers and JEI are ground truth.

How stacking works

Each drawer has several upgrade slots (commonly five). Installed Storage Upgrades combine their multipliers rather than just taking the highest, so a drawer with multiple high-tier upgrades reaches numbers in the tens or hundreds of thousands of items per slot. Filling every slot with the best upgrade you can afford is the standard "this drawer will never fill" move for your most abundant resources.

  • 1 drawer, many upgrade slots
  • Multipliers stack
  • Per-slot, not per-block
  • Reversible — pop the upgrade back out

Creative Storage Upgrade

The Creative Storage Upgrade is a creative-only / endgame item that makes a slot effectively infinite (and in some variants, an infinite source that never empties). Use it for test setups, infinite-resource builds, or as a "tap" that perpetually supplies an item to automation. It is intentionally hard or impossible to obtain in survival without admin help.

03 — Behaviour, not size

Utility Upgrades & Keys

Beyond raw capacity, a set of utility upgrades change how a drawer behaves, and a set of keys are held items you click drawers with to retune them. Mix them with Storage Upgrades in the same upgrade slots.

Installed utility upgrades

Upgrades that change behaviour
UpgradeEffect
Void UpgradeDiscards any items inserted beyond the slot's capacity — the drawer never backs up a pipe. Perfect for junk you over-produce.
Conversion UpgradeLimits or controls equivalent-item conversions (e.g. ore-dictionary swaps), so the drawer doesn't silently accept "the same" item from another mod when you don't want it to.
Redstone UpgradeMakes the drawer emit a redstone signal based on fullness — strength scales with how full it is. Pair with the max/min variants to trigger automation.
Filler UpgradeWorks with automation to keep a drawer topped up / supplied; useful for buffering a continuous feed.
Void + Redstone is the autofarm combo

A drawer with a Void Upgrade on a farm's output never clogs the system, and a Redstone Upgrade lets you switch the farm off (or light a lamp) once the drawer is full so you stop wasting cycles. Together they turn a drawer into a self-managing overflow sink.

Keys — held tools you click with

The key items
KeyWhat it does
Drawer Key / Quantify KeyToggles the count display on the drawer face on or off.
Concealment KeyHides the trim and front detailing, leaving a cleaner, flatter drawer face for seamless builds.
Personal Key / Lock KeyLocks a drawer to its current item(s) so an empty slot keeps its assignment, and/or makes the drawer personal (owner-restricted) so others can't pull from it on a server.
Shroud KeyHides the item icon and count entirely, leaving a blank drawer face — for hidden caches or pure aesthetics.
Why lock a drawer

An unlocked empty drawer forgets its item and will accept the next thing piped in. Locking pins the assignment so a drawer reserved for, say, redstone dust stays "redstone dust only" even at zero count — essential when a controller is auto-sorting a feed across a wall and you don't want stray items landing in the wrong slot.

04 — One wall, one inventory

The Drawer Controller

The Drawer Controller is what turns a pile of drawers into a system. Placed adjacent to a network of connected drawers, it links every drawer within range into one logical inventory, so a single hopper, pipe, or item conduit feeding the controller routes each item to the correct drawer automatically.

How the network forms

The controller reaches out through connected drawers and trim up to a range limit (commonly a 12-block radius / chain depth in default config). Any drawer touching that chain joins the network. Insert an item into the controller and it finds the drawer that already holds that item (or an empty, unlocked slot) and deposits it there; pull from the controller and it draws from wherever that item lives.

Controller components
BlockRole
Drawer ControllerThe brain. One per network; sits flush in the wall and exposes a single insert/extract face for hoppers and pipes.
Controller SlaveAn extender — place it elsewhere on the network to give the controller another I/O point and effectively reach further drawers without a second controller.
TrimCarries the network connection across gaps so framed/bordered walls still link end to end.
The hopper-fed wall

Point a hopper (or any auto-insert) into the Drawer Controller and dump mixed loot in. The controller fans each item to its matching drawer across the whole wall — cobble to the cobble drawer, iron to the iron drawer — no per-drawer piping. Lock your drawers (§03) first so empties don't grab the wrong item.

Mind the range

Drawers beyond the controller's reach silently drop off the network. If a far drawer isn't receiving items, it's usually out of range or the chain is broken by a non-connecting block — add a Controller Slave nearer the gap, or keep the wall within the configured radius.

05 — Tiered metals in one block

Compacting Drawer

The Compacting Drawer is a specialist: it stores a 3-tier compaction group — nugget ↔ ingot ↔ block — in a single drawer and converts between the tiers automatically. Feed it nuggets and they roll up into ingots and blocks; pull whichever tier you need and it breaks back down on demand.

How it works

Drop in any tier of a 9:1-style group (the classic nugget→ingot→block ladder) and the drawer tracks one underlying total while presenting all three tiers on its face. Inserting nine nuggets is the same as inserting one ingot; extracting a block pulls nine ingots' worth. This makes it ideal for the output of an ore/metal pipeline — everything lands in one slot, perfectly compacted, no overflow of half-stacks across tiers.

What a Compacting Drawer is great at
UseWhy
Furnace / smelter outputIngots auto-compact into blocks as they pile up, saving capacity and keeping the count tidy.
Mob/ore farm metalsIron, gold, copper drop in at any tier and unify into one balance.
On-demand crafting supplyPull nuggets, ingots, or blocks of the same metal from one place without manual recipe round-trips.
Pair it with the controller

A Compacting Drawer on a controller network means your whole base can request "one iron block" or "a few nuggets" from a single sorted source, and the drawer does the compaction math for you. It's the cleanest way to present a metal stockpile.

One group per drawer

A Compacting Drawer handles one nugget/ingot/block family at a time and only items that actually form a valid 3-tier craft cycle. Items without a clean reversible compaction recipe won't tier correctly — use a regular drawer for those.

06 — Make it disappear

Framed Drawers

A wall of oak drawers is functional but loud. Framed Drawers let you retexture a drawer with any block in the game, so your storage hall can wear stone bricks, deepslate, quartz, or whatever your build calls for — the drawers vanish into the architecture while keeping every feature.

The Framing Table

Craft the uncoloured Framed Drawer blocks, then take them to a Framing Table. There you assign up to a few materials — typically a side/body block, a trim block, and a front/panel block — and the table stamps that appearance onto the drawer. The result behaves exactly like a normal drawer (slots, capacity, upgrades, controller-compatible) but looks like the blocks you chose.

Framing slots (typical)
SlotControls
Body / side materialThe main face and sides of the drawer block.
Trim materialThe border/frame detailing around the slots.
Front / accent materialThe inner panel or accent, for two-tone looks.
Framed Trim too

The Framing Table also frames Trim and the Compacting/half variants, so an entire seamless wall — drawers, borders, fillers — can share one material and read as a single decorated surface with hidden storage behind every panel.

07 — Patterns & comparison

Tips & Patterns

A few standard setups that make Storage Drawers sing, plus how it stacks up against its modern cousin.

  1. Controller + Compacting on furnace output. Run smelter results into a controller network whose metals live in Compacting Drawers — ingots auto-roll into blocks and a single insert face keeps it tidy.
  2. Void the junk. Put Void Upgrades on cobble, dirt, gravel, and netherrack drawers so over-producing farms never clog — overflow just evaporates once the (huge, upgraded) slot is full.
  3. Lock your assignments. Key-lock every drawer on a sorting wall so empties hold their item and the controller never drops something into the wrong slot.
  4. Redstone the full ones. A Redstone Upgrade on a key drawer can pause a farm or light a "stockpile full" lamp when it tops out.
  5. Frame the whole wall. Once the logistics work, run everything through the Framing Table so the warehouse looks like part of the build, not a grid of crates.

Storage Drawers vs. Functional Storage

Functional Storage is the modern take on the same concept — bulk single-item drawers, capacity upgrades, a controller, compacting drawers, framed/painted drawers. If your pack ships Functional Storage instead of (or alongside) Storage Drawers, your instincts carry over directly.

The two bulk-drawer mods at a glance
AspectStorage DrawersFunctional Storage

NetworkDrawer Controller + Controller SlaveStorage Controller + linking tools

Pick it whenYou want the classic, battle-tested modYour pack already bundles it
Both, not either

Many packs include only one. They solve the same problem, so don't agonize — use whichever your pack ships, and apply every pattern in this guide. For where bulk drawers fit alongside chests, barrels, and full item networks, see the Storage hub.

When in doubt, trust the game

Capacities, multipliers, and the controller's range are all configurable and shift between packs and versions. Treat every number here as an approximate default and let the in-game tooltips and JEI settle any disagreement.

★ — The path

Progression — from basic drawers to a controller wall

Storage Drawers grows one block at a time. Start with a single drawer you can read at a glance, raise its capacity, teach it to manage overflow, then chain a whole wall into one hopper-fed inventory. This is the route from one oak drawer to a self-sorting, compacting, framed storage hall.

  1. Lay the foundation. Place a few Basic Drawers for your most abundant items and hit them with a Drawer Key so each face shows a live count — now your storage is readable without opening anything. See §01.
  2. Raise capacity. Slot Storage Upgrades into each drawer — work up the ladder from I→VI and into Emerald → Diamond → Obsidian as you progress. Multipliers stack, so a fully upgraded slot holds an item count you'll never exhaust. See §02.
  3. Add utility behaviour. Drop a Void Upgrade on over-producing farm sinks so they never clog, and a Redstone Upgrade on key drawers to pause a farm or light a "full" lamp. See §03.
  4. Compact your metals. Swap in a Compacting Drawer for each nugget/ingot/block family so smelter output auto-rolls into blocks and any tier is one pull away. See §05.
  5. Link the wall. Drop a Drawer Controller beside the network (add a Controller Slave to reach farther drawers), then point a hopper at it — mixed loot fans out to the right drawer across the whole wall automatically. See §04.
  6. Make it disappear. Run everything through the Framing Table to build Framed Drawers that wear any block, so the warehouse reads as part of your architecture. See §06.
Storage Drawers vs. Functional Storage

This same progression maps cleanly onto Functional Storage, the modern take on bulk drawers — the muscle memory carries over directly, so follow whichever your pack ships.
