⚒️ Enchantable Tinkers' tools

Reality check — no clean option on 1.19.2

The mods that actually re-add enchanting to Tinkers' tools — Tinkers With Enchants and Tinkers' Enchanting — are 1.20.1 Forge only. The older Pisek's Tinker's Enchantments stops at 1.18.2. There is no confirmed 1.19.2 Forge enchanting addon. So this is the one request we can't satisfy cleanly without changing approach.

Apotheosis workaround

by Shadows_of_Fire

Overhauls enchanting (higher levels, enchantment library, gem/affix system). On 1.19.2 it's the realistic way to pour "enchant-like" power into gear — and it plays with Tinkers' modifier system.

Why: closest you'll get to "enchanted Tinkers tools" on 1.19.2 without a version bump. Big mod — review its loot/affix gen.

Placebo (auto)CurseForge

Tinkers' Levelling Addon best fit

by Sosiska

Tinkers tools gain levels and extra modifier slots as you use them — progression without enchanting.

Why: on 1.19.2 this is the cleanest "make my Tinkers tools stronger over time" mod. Verify the 1.19.2 file (a 1.19.2-1.4.1 build is reported).

needs Tinkers · verify verCurseForge

⚠

Compatibility & performance notes

The stuff worth reading before flipping switches on a server.

Refined Storage vs Applied Energistics 2

Both are endgame digital-storage networks. They run side by side without crashing, but they cover the same niche — running both tends to split progression and let players pick the easier one. My recommendation: keep AE2 as the storage endgame and skip RS, unless you specifically want RS's gentler learning curve as the primary (then consider dropping AE2 to keep identity tight).

Don't stack parallel power/storage systems

We already commit to AE2 + IC2 Classic. The mods that add new parallel systems — Create (rotation), Powah (FE), Tom's Simple Storage (mini-ME) — are fine technically but dilute focus. Treat each as a deliberate "yes we want this pillar," not a default.

Structure spacing

YUNG suite + When Dungeons Arise + Dungeons and Taverns all inject jigsaw structures. If we add the latter two, tune their spacing/separation configs so structures don't overlap-generate (which causes ugly collisions and world-gen lag).

Distant Horizons performance

Heaviest risks are Create (contraptions/block entities) and When Dungeons Arise (large gen at LOD range). Clumps actively helps. Connected Glass/Textures need a Rubidium CTM path to actually render — verify in-world.

Almost Unified scope

It unifies tagged ores/ingots/gems (handy with the Copper Backport), but not decoration or machine recipes. Manage block/recipe overlap (e.g. Quark stones, Chipped vs Rechiseled) via each mod's config, not unification.

Curios synergy (nice thread)

Sophisticated Backpacks, AE2 Wireless Terminals, Comforts and Ars Nouveau all integrate with the Curios we already run — a coherent set of "wear it / quick-access" slots.

Standard install caveat

Before installing, confirm each mod's exact 1.19.2 Forge file on CurseForge and pull required libraries (Moonlight, Balm, Citadel, AutoRegLib, Sophisticated Core) — CurseForge resolves most automatically.
