A field manual for Building Gadgets — the mod that lets you place, swap, copy, and clear large volumes of blocks in a single click. Rough out a tower, retexture a whole wall, save a room and stamp it elsewhere, or carve out a basement in one motion. This guide walks each of the four gadgets, the Construction Paste that feeds the big jobs, and a tidy workflow for using them together.

Building Gadgets MC 1.19.2 · Forge Place · Swap · Copy · Clear Draws from inventory or paste

00 — Orientation

Start Here

Building Gadgets is a set of handheld tools that build, edit, copy, and clear blocks in bulk — drawing the materials straight from your inventory or from Construction Paste. It is the big time-saver for large builds: instead of placing blocks one at a time, you aim, click, and a whole area appears, changes, or vanishes. Treat every mode, size limit, and energy figure here as the typical default and let the in-game radial menu and tooltips win when your pack disagrees.

place

Building Gadget

Set a target block and stamp out an area — columns, walls, stairs, grids, and surfaces — in one click.

swap

Exchanging Gadget

Replace blocks already in the world with a chosen block — instant retexturing without breaking and re-placing.

copy

Copy-Paste Gadget

Capture a structure to a Template, then paste it anywhere; materialises blocks from Construction Paste.

clear

Destruction Gadget

Clear a configurable box of blocks at once; overflow lands back in your inventory.

Two things to learn first

Every gadget shows a ghost preview of what it will do before you commit, and most settings live in a radial / right-click menu (mode, range, fuzzy, anchor). Materials come from your real inventory for normal building; the Copy-Paste gadget instead needs Construction Paste (§05) to fill in blocks.

How it fits the rest of the build hub

Building Gadgets is the placement half of a decoration workflow — it moves blocks fast but does not invent new block styles. Pair it with a palette mod such as Chipped for variety, and with the wider Building & Decoration hub for the trim, frames, and details you stamp once the rough shape exists. See the workflow section (§07) for how the gadgets hand off to each other.

The shape of a job

  1. Pick the gadget for the task — build, swap, copy, or clear.
  2. Set the target block (or copy a structure) by sneak-clicking a block in the world.
  3. Choose a mode and size from the radial menu — column, wall, stairs, grid, surface, and the range/radius.
  4. Aim and click. The ghost preview shows the result; click to commit, consuming blocks or paste.
  5. Undo if needed. The building gadgets support an undo of your last placement while the area is still loaded.

01 — Place in bulk

The Building Gadget

The Building Gadget is the workhorse: set a target block, choose a mode, dial in the range / size, and place a whole area against the face you are pointing at — all in one click. It pulls the real blocks from your inventory (or Construction Paste, if configured), so a wall of 200 stone bricks costs 200 stone bricks.

Setting the block & previewing

Sneak + right-click a block in the world to copy it as the gadget's current block to place; a ghost preview then follows your crosshair. The preview is exactly what will be placed, so you can line it up before committing. Right-click (no sneak) places.

Modes

Building Gadget modes (names & behaviour vary by version)
ModeWhat it builds
Build to MeExtends a line of blocks from the face you target up to where you are standing/looking — fast for spans and bridges.
Vertical ColumnA pillar straight up (or down) from the target face, height set by range.
Horizontal ColumnA flat row extending out from the target face along your facing.
WallA flat rectangle on the targeted face — the bread-and-butter mode for walls and floors.
StairsA stepped diagonal run — instant staircases up or down.
GridA spaced lattice of blocks (e.g. pillars or supports) across the area.
SurfaceCoats the connected surface you point at — re-skins a floor or follows terrain.

Range, size & fuzzy

  • Range / size — set in the radial menu; larger areas cost more blocks (and energy/durability) per click and may hit a config-limited maximum.
  • Fuzzy mode — when on, placement ignores the exact block you targeted and fills based on the area rather than matching only identical neighbours.
  • Connected / overlay options — control whether placement hugs only matching surfaces or spans across.
Stock the bag first

The gadget places real blocks, so keep a deep stack of your material on you (or in a connected inventory if your pack supports it). Run dry mid-click and only part of the area fills — top up and click again to finish.

Mind the maximums

Most packs cap how many blocks a single click can place (commonly in the low hundreds) and how far the range reaches. If a big wall only half-builds, you have hit the per-action limit — split it into a few clicks, or raise the limit in the config if allowed.

02 — Swap in place

The Exchanging Gadget

The Exchanging Gadget swaps existing blocks in an area for a chosen block, without you breaking and re-placing anything. Point at a cobble wall, set the target to polished blackstone, click — the whole face retextures, and the old blocks return to your inventory. The mode set mirrors the Building Gadget, so the area you exchange is shaped the same way (wall, surface, column, and so on).

How it works

  1. Set the replacement block the same way as the Building Gadget — sneak + right-click the block you want to swap to.
  2. Choose a mode & size (Wall and Surface are the usual picks for retexturing).
  3. Aim at the surface to change — the preview highlights which blocks will be swapped.
  4. Click. Matching blocks become the new block; the removed blocks drop into your inventory.
Fuzzy makes whole-wall swaps easy

With fuzzy mode on, the exchanger swaps every block in the targeted area regardless of type — perfect for re-skinning a mixed wall in one pass. With it off, only blocks matching the one you pointed at are exchanged, which is handy for surgical edits (e.g. only the cobble, leaving the windows).

It still costs the new blocks

Exchanging consumes the replacement material from your inventory just like building, and returns the old blocks to you. Make sure you carry enough of the new block to cover the whole area, or the swap stops partway.

03 — Capture & stamp

The Copy-Paste Gadget

The Copy-Paste Gadget captures a structure into a Template, then pastes that design elsewhere — even in another location or world. It is how you build a perfect room once and stamp it ten times. Unlike the building gadget, pasting materialises blocks from Construction Paste (§05) rather than placing each block from your inventory.

Copying a structure

  1. Switch to Copy mode in the radial menu.
  2. Mark two corners of the region you want to capture — the selection box highlights the volume.
  3. Confirm the copy. The layout (blocks and positions) is stored to the gadget's current Template.
  4. Save it if you want to keep it — name the design in the Template Manager so you can reuse it later.

Pasting a structure

  1. Switch to Paste mode and aim where the structure's anchor should land.
  2. Position the ghost preview — nudge/rotate the placement so it lines up with your build.
  3. Make sure you have paste & power. Pasting draws Construction Paste to materialise the blocks (see §05).
  4. Click to paste. The full structure appears at once; use undo if the alignment was off.
Storing and managing designs
Item / screenRole
TemplateThe captured design data the gadget pastes from — the gadget holds one "active" copy at a time.
Template ManagerA block/GUI for naming, saving, loading, and swapping stored designs between gadgets and Paste Containers.
Paste ContainerHolds the Construction Paste the paste consumes; tiers store more (see §05).
Copy is layout, not loot

A Template records the blocks and their arrangement, not tile-entity contents — chests come back empty, machines lose their inventories/settings, and some modded blocks may not copy cleanly. Test-paste a saved room in a spare area before relying on it for a big stamp.

04 — Clear in bulk

The Destruction Gadget

The Destruction Gadget clears a configurable box of blocks in one click — ideal for digging out a basement, levelling a hill, or clearing a botched build. You set how deep, wide, and tall the box reaches; the ghost outline shows exactly what will be removed before you commit.

Using it

  1. Open the settings and set the box dimensions — depth into the face plus the width/height around your aim point.
  2. Aim at the face where the box should start — the outline previews the volume.
  3. Sneak + click (the usual safety gesture) to remove the whole box at once.
  4. Collect the drops. Mined blocks go to you; what won't fit overflows to the world or back into your inventory depending on config.
Overflow goes to your inventory

Cleared blocks are returned rather than vaporised — by default they flow into your inventory (overflow drops nearby when you are full). That makes the Destruction Gadget a fast, mostly non-destructive way to harvest a volume, not just delete it.

It removes a lot, fast

A large box can erase a big chunk of terrain — and your build — in a single click. Keep the dimensions modest until you trust the preview, watch the energy/durability cost (big boxes are expensive), and remember the gadget respects whatever block-protection / claim rules your server enforces.

05 — The fuel for big builds

Construction Paste & Containers

Construction Paste is the generic "filler" the gadgets use to materialise blocks when you are not placing from a literal inventory stack — above all when pasting a copied structure. It lets one bucket of paste stand in for hundreds of varied blocks, which is why copy-paste and very large builds lean on it.

Making paste

Construction Paste is produced from cheap bulk material (commonly via a Construction Chamber or a crafting/processing recipe in your pack) and measured as a volume of paste rather than as item stacks. You feed the raw input in, and the paste accumulates in a container ready for the gadget to draw on.

Where paste lives
Item / blockRole
Construction PasteThe consumable "block material" the Copy-Paste gadget spends to render a pasted structure.
Construction ChamberThe block that produces paste from raw input (recipe/throughput is pack-dependent).
Paste Container (tiered)Stores paste for the gadget to draw on; higher tiers hold a larger volume so you refill less often.

Why builds rely on it

  • Copy-Paste needs it. A pasted structure is materialised from paste, not from your inventory — no paste, no paste-job.
  • It abstracts the bill of materials. One pool of paste covers a structure made of many different blocks, so you do not have to carry every block type the design uses.
  • It scales. Big repeated stamps consume a lot of paste fast; tiered Paste Containers keep enough on hand to finish in one go.
Carry a topped-up container

Before a big paste, check the Paste Container's level — running dry mid-paste leaves a half-built structure. Keep a high-tier container charged from your Construction Chamber so a large stamp lands in a single click.

06 — Keeping them running

Energy & Durability

Every gadget has a running cost. Depending on your pack's config, a gadget either consumes Forge Energy (FE) from an internal battery or wears down a durability bar as you use it. Bigger actions — large walls, deep destruction boxes, full structure pastes — cost proportionally more.

Cost model (config-dependent)
AspectHow it behaves
Energy (FE) modeThe gadget holds an FE buffer; each block placed/removed drains some. Recharge on any FE charger, energy cube, or chargepad.
Durability modeIf configured without FE, the gadget loses durability per action and is repaired/recharged through pack-specific means.
Cost scalingRoughly proportional to blocks affected — a 200-block wall costs far more than a 10-block edit; the Destruction box scales with its volume.

Charging

In energy mode, drop the gadget in any block that outputs FE — an energy cube, a charger, or a chargepad-style block from your tech mods. Many players keep a charger by the build site so the gadget tops up between sweeps. If your pack runs durability mode instead, follow its repair path (anvil, recharge block, or config-set behaviour).

Range upgrades

Building Gadgets can be enhanced with range upgrades that raise the maximum size/reach of an action — letting one click cover a bigger wall or a longer column. Larger actions naturally draw more energy or durability per click, so a stronger power supply pairs well with a higher range cap.

Check which mode your pack uses

FE-vs-durability is a config switch, and packs differ. If your gadget shows an energy bar, keep a charger handy; if it shows durability, learn its repair route. Either way, an empty gadget simply stops placing mid-action — refuel and click again to finish.

07 — Putting it together

Workflow & Tips

The gadgets are strongest as a chain. Rough the shape, retexture it, save the good bits, and clean up the mistakes — each tool handing the build to the next. Here is a workflow that scales from a single tower to a whole town.

  1. Rough the shape with the Building Gadget. Use Wall and Column modes to block out walls, floors, and pillars fast in a cheap placeholder block.
  2. Retexture with the Exchanging Gadget. Once the massing looks right, swap the placeholder for your real palette — whole faces at a time, with fuzzy mode for mixed walls.
  3. Save reusable rooms with Copy-Paste. Capture a finished room, window, or arch to a named Template, then stamp it wherever it repeats — keeping a charged Paste Container on you.
  4. Clear mistakes with the Destruction Gadget. Carve out interiors, fix overshoots, and reclaim the blocks back into your inventory.

Pairing with palette mods

Building Gadgets moves blocks; it does not create new looks. Pair it with Chipped (and similar palette mods) for the variety — design a wall in a Chipped variant, then use the Exchanging Gadget to roll that texture across the whole structure in seconds. The gadgets are the muscle; the palette mod is the wardrobe.

  • Build rough massing
  • Exchange the palette
  • Copy reusable rooms
  • Destroy to fix & reclaim
  • Paste needs paste

    • Handy habits

    • Watch the preview. The ghost outline is ground truth for what a click will do — line it up before committing, especially with Destruction and Paste.
    • Use undo. The last building action can usually be reversed while the area is loaded — cheaper than re-clearing by hand.
    • Keep fuel topped up. Energy/durability and Construction Paste all run out mid-action and leave a job half-done; refill, then re-click.
    • Respect server rules. On multiplayer, gadgets honour claims/protection and any size limits the admins set.
    Exact modes & limits vary by version

    Mode names, the maximum blocks per click, range caps, the FE-vs-durability switch, and the paste recipe all change between Building Gadgets versions and packs. Use this guide for the shape of the workflow, and let the in-game radial menu, tooltips, and JEI settle the specifics for your install.
